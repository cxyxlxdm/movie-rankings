# 灵魂之镜题目生成规格

## 任务
生成 1000 道"灵魂之镜"二选一题目，每道题的 prompt 必须根据具体的 A/B 选项定制（不是随机模板）。

## 输出文件
`/Users/liang/movie-rankings/public/soul-mirror-questions-v2.json`

## JSON 格式
```json
[
  {
    "a": "选项A",
    "b": "选项B", 
    "category": "类别",
    "prompt": "根据具体选项定制的提问，A 还是 B"
  }
]
```

## 类别分布（共 1000 题）
| 类别 | 数量 | 说明 |
|------|------|------|
| 电影 | 120 | 经典电影两两对比 |
| 人物 | 120 | 历史/文化/科学家等人物对比 |
| 文学 | 100 | 经典文学作品对比 |
| 导演 | 80 | 导演风格/作品对比 |
| 游戏 | 100 | 经典游戏对比 |
| 电视剧 | 100 | 经典剧集对比 |
| 音乐 | 100 | 歌手/乐队/专辑对比 |
| 历史 | 100 | 历史事件/朝代对比 |
| 诗词 | 80 | 经典诗句对比 |
| 科技 | 50 | 科技产品/发明对比 |
| 哲学 | 50 | 哲学思想/流派对比 |

## 核心规则

### 1. prompt 必须跟选项内容匹配
- ❌ "俄罗斯方块 vs 贪吃蛇 → 哪个美术风格更让你着迷"（这俩没美术风格）
- ✅ "俄罗斯方块 vs 贪吃蛇 → 哪个更考验你的瞬时反应"
- ❌ "魏征 vs 长孙无忌 → 谁的诗更让你想听"（都不是诗人）
- ✅ "魏征 vs 长孙无忌 → 谁的谏言更让你佩服"

### 2. 同类别内 prompt 必须多样化
- 同一类别的 100 题中，同一个 prompt 模板最多用 2 次
- 人物题不能全用"你更亲近哪一个"
- 游戏题不能全用"如果通宵只能玩一个"

### 3. prompt 风格参考（根据内容变化）
- 场景型："如果可以穿越到那个时代"、"如果只能教孩子一首"
- 感受型："哪个结尾更让你怅然若失"、"谁的命运更让你感叹造化弄人"
- 选择型："如果余生只能保留一个"、"如果要给最爱的人推荐一部"
- 假设型："如果你活在那个时代会追随谁"、"如果你是导演会拍哪个故事"
- 共情型："哪个更像你此刻的心境"、"谁的孤独你更能理解"

### 4. 题目质量
- 选项要有可比性（同类型、同级别）
- 避免太冷门的选项（大众认知度要高）
- 避免重复（同一对不能出现两次）
- 选项用中文
